Requisitos de Aplicación

Descripción de la Aplicación

Esta aplicación permitirá visualizar, a través de un mapa interactivo, la relación entre los niveles de educación de diferentes zonas del área de Greater London (alrededores de Londres) y la cantidad de colegios presentes en cada zona. Utilizando datos demográficos y educativos, la aplicación mostrará patrones y correlaciones entre el nivel educativo promedio de cada zona y el número de instituciones educativas disponibles.

La aplicación está dirigida a investigadores, educadores, y responsables de la planificación educativa para ayudarles a comprender mejor cómo la distribución de los colegios influye en los niveles de educación en diferentes áreas de Greater London.

Características de la Aplicación

  • Mapa interactivo con diferentes capas de visualización para ver áreas de Greater London.
  • Filtros para seleccionar zonas específicas y niveles de educación.
  • Estadísticas detalladas sobre la relación entre número de colegios y niveles educativos en cada zona.
  • Herramienta de análisis para comparar zonas y observar tendencias a lo largo del tiempo.

Requisitos Técnicos

  • Conexión a Internet.
  • Navegador web actualizado (Chrome, Firefox, Safari o Edge).
  • Soporte para JavaScript habilitado para una experiencia interactiva.
